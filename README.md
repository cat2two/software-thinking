# Stock Simulation Investment Project

가상의 자금을 이용하여 주식 매매를 체험할 수 있는 주식 모의 투자 프로그램입니다.  
사용자는 주식을 매수 및 매도하며 자산 변화를 확인할 수 있고, 투자 흐름과 기본적인 금융 시스템을 학습할 수 있습니다.

---

## Project Overview

본 프로젝트는 실제 주식 투자 환경을 간단하게 구현한 모의 투자 시스템입니다.  
사용자는 가상 자금을 지급받아 원하는 종목을 거래할 수 있으며, 거래 결과에 따른 수익률과 보유 자산을 확인할 수 있습니다.

---

## Main Features

- 플레이어 추가
- 가상 자금 지급
- 주식 매수 및 매도 기능
- 현재 보유 자산 조회
- 수익률 계산
